# 🌍 World Cup AI Predictor

An end-to-end Machine Learning project that predicts international football match outcomes using historical match data.

## Tech Stack

- Python
- Pandas
- NumPy
- Scikit-Learn
- Jupyter Notebook

## Dataset

Historical international football matches from Kaggle.

## Features Used

- Home Team
- Away Team
- Neutral Venue
- Tournament Importance
- Matches Played
- Win Rate
- Average Goals Scored
- Average Goals Conceded
- Recent Form
- Goal Difference

## Model

Random Forest Classifier

## Current Performance

- Accuracy: **55.7%**
